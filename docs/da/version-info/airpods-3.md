# AirPods 3

::: tip
Denne produktlinje opdateres ikke længere aktivt af producenter, da udviklingen er skiftet til nyere modeller. AirPods 3-kopier er stadig fremragende produkter og forbliver populære valg. Hvis du leder efter de nyeste funktioner og forbedringer, overvej at tjekke [AirPods 4](/da/version-info/airpods-4).
:::
