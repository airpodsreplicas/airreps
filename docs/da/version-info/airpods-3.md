# AirPods 3

::: tip
Denne produktlinje opdateres ikke længere aktivt af producenterne, da udviklingen er flyttet til nyere modeller. AirPods 3-replikker er stadig fremragende produkter og forbliver populære valg. Hvis du leder efter de nyeste funktioner og forbedringer, så overvej at se [AirPods 4](/da/version-info/airpods-4).
:::
