# AirPods 3

::: tip
Diese Produktlinie wird von den Herstellern nicht mehr aktiv weiterentwickelt, da die Entwicklung auf neuere Modelle verlagert wurde. AirPods 3-Repliken sind weiterhin ausgezeichnete Produkte und bleiben beliebte Optionen. Wenn Sie nach den neuesten Funktionen und Verbesserungen suchen, schauen Sie sich [AirPods 4](/de/version-info/airpods-4) an.
:::
