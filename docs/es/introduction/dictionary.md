# Diccionario

## W2C: Dónde comprar / Dónde conseguir

"W2C" significa "¿Dónde puedo comprar este artículo?" Se usa cuando se busca la fuente o el vendedor de un producto.

**Ejemplo:** "¿Puedes poner el W2C de los V5.3's en el subreddit?"

## Reps/Clones: Réplicas

"Reps" o "Clones" se refieren a imitaciones de productos diseñadas para copiar un producto en particular (en este caso, AirPods) lo más precisamente posible. Esto se distingue de una 'knockoff'.

**Ejemplo:** "¿Dónde puedo conseguir las mejores reps/clones de AirPods?"

## Fake AirPods

"Fake AirPods" es un término informal para réplicas o clones de AirPods. En esta guía, usamos "réplicas" o "clones" para describir productos de terceros de mayor calidad que intentan igualar las funciones de Apple a un precio inferior.

## Knockoffs

"Knockoffs" se refiere a productos diseñados para parecerse a marcas populares evitando infringir la propiedad intelectual (por ejemplo, "Gucce" en lugar de "Gucci").

## GP: Conejillo de indias

"Hacer de Guinea Pig" significa ser la primera persona en pedir y reseñar el producto de un vendedor, permitiendo que la comunidad evalúe su calidad.

**Ejemplo:** "¿Alguien está dispuesto a GP los V4 de Troy?"

## QC: Control de calidad [FOTOS]

"QC" significa Quality Control. "QC PICS" se refiere a las fotos utilizadas para verificar la calidad de un producto comprado. Consulta nuestra [Guía de QC y LC](/es/ordering/qc-lc) para saber cómo inspeccionar tus réplicas.

**Ejemplo:** "¿Puedes enviarme las QC pics de los HiCity v2.5's?"

## LC: Verificación de autenticidad

Un "Legit Check" verifica la autenticidad de un producto, a menudo usado al comprar en fuentes no minoristas como eBay o Facebook Marketplace.

**Ejemplo:** "¿Puede alguien LC estos AirPods Pros por mí?"

## Agent: Agente de carga / Transitario

Un "Agent" es un tercero que facilita compras seguras desde vendedores (a menudo en China) hacia compradores. Actúan como representantes, asegurando que los productos no estén defectuosos y que los compradores reciban lo que pidieron. Recomendamos [KakoBuy](https://airreps.link/kakobuy) — consulta nuestra [guía de KakoBuy](/es/ordering/kakobuy) para saber cómo usarlo.

**Ejemplo:** "Puede que necesites usar un agente para conseguirlos, ya que actualmente HiCity no envía a Filipinas."

## Markings: Etiquetado del producto

"Markings" se refiere al texto en la funda y los AirPods que proporciona información sobre el producto, incluyendo el texto "Designed by Apple in California", números de modelo y números de serie. Para más sobre cómo varían los markings entre réplicas y originales, consulta la [calidad de construcción](/es/introduction/build-quality).
