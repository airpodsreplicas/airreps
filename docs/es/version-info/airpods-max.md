## Diferencias de Versión V2 y V2.5:

**V2 TB**: El modelo V2 ofrece calidad de sonido satisfactoria con soporte para ajustes de EQ. Cuenta con una carcasa más firme y ofrece preparación para el futuro a través de capacidad de actualización de firmware.

**V2.5**: Esta versión mejorada proporciona Cancelación Activa de Ruido (ANC) mejorada y modo de transparencia mejorado. También cuenta con un diseño de diadema superior comparado con el V2. Aunque se anuncia que soporta actualizaciones de firmware, no se han lanzado actualizaciones para este modelo hasta ahora (para la versión Bluetrum 893x);
Realtek 8763ESE sí soporta OTA.

## Visión General V3

V3 es una mejora significativa respecto al V2.5, con un chipset diferente (Airoha 1562AE) y características mejoradas incluyendo ANC Híbrido mejorado, Audio Espacial con Seguimiento de Cabeza y opciones de construcción de metal. La versión V3 Metal ofrece copas de oído y diadema de metal premium, mientras que la versión V3P usa construcción de plástico.
