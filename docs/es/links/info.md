## Preguntas Frecuentes

::: details ¿Recibiré una unidad de buena calidad?
No es posible garantizar la calidad de una unidad, ya que puede variar incluso entre unidades del mismo vendedor. Sin embargo, los vendedores mencionados en el texto original son generalmente conocidos por ofrecer productos de buena calidad.
:::

::: details ¿Cómo contacto a los vendedores?
Para contactar vendedores, puedes unirte al Discord de AirReps: https://airreps.link/discord y obtener sus números de WhatsApp, ya que no están disponibles públicamente. Alternativamente, puedes usar AliExpress para enviarles mensajes haciendo clic en "Contact Supplier".
:::

::: details ¿Cuál es el mejor vendedor?
El mejor vendedor no se puede determinar definitivamente ya que varía según tus necesidades y la plataforma desde la que elijas comprar. Los vendedores típicamente ofrecen productos similares, por ejemplo, el 'Pro 2 V5.2 TB' de Jenny es idéntico al de HiCity. Sin embargo, surgen diferencias en aspectos como soporte, tiempos de envío y las plataformas a través de las cuales se realizan las compras.
:::

::: details Ordené hace x días y mi paquete no ha llegado. ¿Qué debo hacer?
El envío desde China a veces toma tiempo. La información de seguimiento puede desaparecer una vez que el paquete sale de China, y solo se actualizará cuando llegue a aduanas en tu país. Si ha pasado mucho tiempo y aún no hay información de seguimiento disponible, deberías contactar a tu vendedor para una actualización.
:::

::: details Mi información de seguimiento está en chino. ¿Qué debo hacer?
Si tienes información de seguimiento en chino, puedes copiar el número de seguimiento y pegarlo en el sitio web [17Track](https://www.17track.net/). Esto te permitirá ver actualizaciones en inglés. Alternativamente, puedes abrir las actualizaciones de seguimiento en una computadora y pegarlas en Google Translate para verlas en un idioma diferente.
:::

::: details Mi tarjeta no funciona. ¿Qué debo hacer?
Si tu tarjeta no funciona, deberías contactar al vendedor y preguntar si aceptarán una forma diferente de pago. Ten en cuenta que algunos métodos de pago alternativos, como Wise, no ofrecen protección al comprador. Si el vendedor no puede aceptar una forma diferente de pago, puedes intentar usar una tarjeta diferente o comprar de un vendedor diferente.
:::

::: details ¿Por qué soy redirigido a un sitio web de crypto al pagar con Tarjeta de Crédito?
Los vendedores de confianza usan una pasarela de pago crypto segura para procesar pagos con Tarjeta de Crédito. Tu tarjeta se usa para comprar crypto, que luego se usa para completar tu pedido de forma segura y eficiente.

**Nota: Algunos procesadores de pago pueden ocasionalmente solicitar una verificación de ID (KYC) para cumplir con regulaciones locales, aunque esto no siempre es requerido.**

**Alternativa:** Si quieres evitar cualquier posibilidad de verificación KYC, considera usar [OOPBuy](https://airreps.link/oopbuy), una plataforma de agentes que compra productos de vendedores en tu nombre y proporciona fotos de control de calidad (QC) antes del envío. OOPBuy usa procesadores de pago regulares que aceptan tarjetas de crédito y débito y **nunca requieren verificación KYC** o verificaciones de identidad. Esto lo hace una excelente opción si prefieres usar tu tarjeta sin ningún requisito de KYC mientras aún obtienes la seguridad adicional de fotos QC.
:::

::: tip
Algunos sitios web de vendedores que proporcionan pagos con tarjeta de crédito pueden tener branding discreto del sitio web. Busca el número de versión que quieres - puedes ignorar con seguridad las imágenes genéricas de productos. Esto se hace para cumplir con políticas de copyright que algunas pasarelas de pago con tarjeta de crédito pueden tener.
:::
