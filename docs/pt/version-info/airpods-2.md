# AirPods 2

::: tip
Esta linha de produtos não está mais sendo atualizada ativamente pelos fabricantes, pois o desenvolvimento foi direcionado para modelos mais novos. Réplicas do AirPods 2 ainda são produtos excelentes e continuam sendo escolhas populares. Se você procura os recursos e melhorias mais recentes, considere conferir [AirPods 3](/pt/version-info/airpods-3) ou [AirPods 4](/pt/version-info/airpods-4).
:::
