# AirPods 2

::: tip
Esta linha de produtos não está mais sendo atualizada ativamente pelos fabricantes, pois o desenvolvimento mudou para modelos mais novos. As réplicas de AirPods 2 ainda são produtos excelentes e continuam sendo escolhas populares. Se você procura os recursos e melhorias mais recentes, considere conferir os [AirPods 3](/version-info/airpods-3) ou [AirPods 4](/version-info/airpods-4).
:::
