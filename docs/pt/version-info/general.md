# Geral

::: tip Sobre os Números de Versão
Este sistema de numeração de versões foi projetado pela comunidade AirReps para simplificar a identificação de produtos entre diferentes fabricantes. Quando novos modelos de AirPods são lançados (como AirPods Pro 2 ou Pro 3), a numeração de versão continua a partir dos números anteriores em vez de reiniciar.

Fabricantes como TigerBuilder e Huilian usam seus próprios nomes de produto (como "377super"), o que pode dificultar a comparação de produtos entre fabricantes. Nosso esquema de nomenclatura da comunidade resolve isso fornecendo um número de versão padronizado que facilita encontrar produtos comparáveis entre fabricantes sem ter que lembrar dezenas de nomes diferentes.

**Nota:** Fabricantes não são o mesmo que vendedores. Você não pode comprar diretamente dos fabricantes — você compra de vendedores que obtêm produtos desses fabricantes.
:::

::: details Por que comunidades semelhantes podem ter produtos mais novos do que este guia?
Os produtos listados neste guia são da mais alta qualidade e são cuidadosamente selecionados para atender aos nossos padrões. Atualizamos a lista diariamente para garantir que apenas os melhores produtos sejam incluídos. Outras comunidades podem ter padrões mais baixos e podem incluir produtos que não são tão bem feitos. Também é possível que usem um esquema de nomenclatura diferente para seus produtos, o que pode fazer parecer que eles têm produtos mais novos do que este guia.
:::

::: details Qual modelo é o melhor?
O modelo listado no topo é o mais recente, enquanto os outros modelos são versões anteriores. Para encontrar o melhor para você, considere fatores como seu orçamento, disponibilidade de envio e recursos desejados. Também é uma boa ideia procurar avaliações dos diferentes modelos e tomar sua própria decisão com base nessas informações.
:::

::: details Como posso determinar a versão de um clone que encontrei?
Não é possível determinar a versão de um clone sem realmente usá-lo. Os números dos chips e as fotos fornecidas pelos vendedores não são indicadores confiáveis da versão de um clone. Perguntar aos vendedores se um clone específico é certa versão não fornecerá informações úteis, pois eles podem não entender a pergunta. Se você estiver disposto a correr o risco, pode comprar um clone aleatório. No entanto, para garantir que saiba exatamente qual modelo está recebendo, é melhor comprar de vendedores verificados.
:::

::: details Como posso identificar o chip nos meus AirPods?
Para identificar o chip nas suas réplicas de AirPods, consulte esta lista de [apps úteis](/pt/useful-apps). Teste cada app individualmente para determinar com qual eles se conectam. Isso ajudará a identificar rapidamente o fabricante e o chipset. Observe que, se seus AirPods forem de nível muito baixo, eles podem não se conectar a nenhum dos apps. Nesses casos, eles provavelmente usam o chipset Jieli.
:::

::: details Como posso determinar se meus AirPods são reais?
Para verificar rapidamente se seus AirPods são reais, conecte-os a um iPhone e vá em Ajustes > Geral > Sobre. Se houver uma seção "AirPods" ao rolar para baixo, verifique se o número de série mostrado lá corresponde ao número de série dentro da tampa dos AirPods ([Check Here](http://airreps.info/aboutsection)). Se não houver seção AirPods, então seus AirPods provavelmente são falsos. Algumas réplicas mais novas podem ter uma página "Sobre" funcional; nesse caso você pode verificar a presença de Spatial Audio e rastreamento de cabeça. Para AirPods, você também pode verificar a versão do firmware na página Sobre para garantir que corresponda à versão mais recente lançada pela Apple. Se não tiver certeza sobre a autenticidade dos seus AirPods e não puder verificar antes da compra, você pode pedir ajuda no [r/airreps subreddit](https://www.reddit.com/r/AirReps/) ou no [Discord](https://airreps.link/discord).
:::

::: details i9999, Superpods, etc. são melhores do que Supercopy ou os produtos listados no guia?
No passado, esses nomes eram usados para indicar upgrades ou melhorias às réplicas de AirPods. No entanto, vendedores chineses começaram a usar esses nomes para dar a impressão de que estão oferecendo produtos mais novos ou melhores quando, na verdade, estão apenas vendendo os mesmos produtos de antes. Como resultado, esses nomes não têm mais significado e devem ser ignorados. Em vez disso, foque nas características e na qualidade reais dos produtos ao tomar uma decisão de compra.
:::
