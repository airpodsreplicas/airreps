# AirPods 3

::: tip
Эта линейка продуктов больше не получает активных обновлений от производителей, поскольку разработка сместилась на более новые модели. Реплики AirPods 3 по-прежнему остаются отличными продуктами и остаются популярным выбором. Если вы ищете последние функции и улучшения, рассмотрите возможность ознакомиться с [AirPods 4](/ru/version-info/airpods-4).
:::
