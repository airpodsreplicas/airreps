# AirPods Pro

::: warning Больше не продаются у проверенных продавцов
Оригинальная реплика **AirPods Pro (1st generation)** была заменена и **больше не продаётся у проверенных продавцов** — она просто слишком устарела, и сообщество перешло на более новые и гораздо более функциональные модели.

Если вы хотите реплику в стиле Pro, возьмите одну из этих моделей вместо неё:

- **[AirPods Pro 2 реплики](/ru/links/airpods-pro-2)** — наиболее зрелый и функционально полный вариант Pro.
- **[AirPods Pro 3 реплики](/ru/links/airpods-pro-3)** — самая новая модель с самыми новыми функциями.
:::

Хотите посмотреть, как отличались старые версии AirPods Pro? Страница [Информация о версиях AirPods Pro](/ru/version-info/airpods-pro) сохранена для справки.
