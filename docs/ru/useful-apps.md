# Полезные приложения

Обновление прошивки на ваших репликах часто может добавить новые функции, такие как пространственное аудио, или улучшить их стабильность. Для этого вы можете использовать различные приложения, доступные для загрузки. Однако обратите внимание, что **большинство этих приложений доступны только для устройств Android**.

Ниже представлены приложения, которые вы можете загрузить, а также версии и чипсеты реплик, которые они поддерживают. Если у вас возникнут проблемы с установкой приложений не из Play Store, убедитесь, что в настройках включена опция «Установка из неизвестных источников» (в разделе «Безопасность» или «Специальные возможности»). Для получения дополнительной информации о том, как использовать эти приложения и избежать проблем, посетите наш [Discord](https://airreps.link/discord).
